# 家事模擬器 Housework Simulator

把一連串家事變成清楚、可完成的回合。選好任務後，畫面中央會顯示目前要做的家事，外圈則以圓形讀條倒數；完成全部任務後，再透過四級家事檢核表確認成果。

## 功能

- 三大類家事：清潔類、整理類、僕人類
- 12 個預設任務，可自由複選
- 每項任務可設定 1、3、5、10、15 或 20 分鐘
- 中央家事提示搭配 SVG 圓形倒數讀條
- 輕按兩下中央計時區可提早完成並進入下一項
- 支援暫停、繼續、重新計時與按鈕完成
- 每項家事都有專屬的四級成果描述
- 完成摘要與最近 20 次紀錄（儲存在瀏覽器 localStorage）
- 支援手機震動提示、鍵盤操作、PWA 離線快取
- 不需安裝套件，也不需後端

## 使用方式

直接開啟 `index.html` 即可使用。為了啟用 Service Worker 與完整 PWA 功能，建議透過本機靜態伺服器啟動：

```bash
python -m http.server 8000
```

接著開啟 `http://localhost:8000`。

## 自訂家事

所有家事資料都集中在 `app.js` 的 `TASK_CATEGORIES`。每個任務包含：

- `name`：家事名稱
- `icon`：顯示圖示
- `short`：選擇頁短說明
- `prompt`：計時頁中央提示
- `ratings`：由最好到最差的四級檢核描述

加入新任務時，沿用同一資料格式即可自動出現在選擇頁與檢核表。
